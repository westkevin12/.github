# Code of Conduct

## Goal

Our goal is to build high-quality software. We value technical excellence, constructive feedback, and professional collaboration.

## Standards

- **Focus on the Technical:** Discussions should remain focused on code, architecture, and solving problems.
- **Be Professional:** Treat fellow developers with respect. Criticism of code is expected and encouraged; personal attacks are not.
- **No Harassment:** We have a zero-tolerance policy for harassment of any kind.

## Enforcement

If you witness or experience behavior that violates these standards, Ignore it and write more code.
